# Mathematical Operators
Basic arithmetic: +, -, *, /

Use parentheses () to control order of operations (PEMDAS)

## Example

    a = 10
    b = 5

    print(a + b)  # 15
    print(a - b)  # 5
    print(a * b)  # 50
    print(a / b)  # 2.0

    # Average with parentheses
    avg = (5 + 7 + 9) / 3
    print(avg)  # 7.0
